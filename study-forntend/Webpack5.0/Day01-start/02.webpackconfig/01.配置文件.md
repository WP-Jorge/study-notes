### 1、配置文件常见配置项
```shell
# mode (模式) 必要

# entry (入口) 一般不能省略

# output (出口) 一般也是不能省略的

# module (模块配置，各种不同的文件的处理的配置都在这里 ---- 各种loader配置) 

# plugin (插件)

# devService (开发服务器的相关配置)
```

### 2、创建webpack.config.js文件进行相关配置
```js
/**
 * webpack.config.js
 * webpack的配置文件
 */
const { resolve } = require('path')

module.exports = {
	// 打包模式
	mode: 'development', // 开发模式
	
	// 入口文件
	entry: './src/index.js',
	
	// 出口文件
	output: {
		// 输出目录（输出目录必须是绝对路径）
		path: resolve(__dirname, 'dist'), // 当前变量下的dist目录
		
		// 输出文件的名称
		filename: 'main.js'
	}
}
// 配置完成后可以直接在命令行输入webpack进行打包
```
