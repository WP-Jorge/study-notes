**1、哨兵机制**

- 在主节点创建哨兵机制配置

  - 在root目录下创建sentinel目录

    ``mkdir sentinel``

  - 进入目录创建sentinel.conf

    ``cd sentinel``

    ``touch sentinel.conf``

  - 配置sentinel.conf配置文件

    ``vim sentinel.conf``

    添加

    ``sentinel monitor 被控制数据库的名字（自己的名字） ip port 1(表示只要有一个投票同意就转换主数据库)``

    ``bind 0.0.0.0``开启远程访问

  - 启动哨兵模式进行测试

    ``cd /usr/redis/bin``

    ``redis-sentinel /root/sentinel/sentinel.conf``
